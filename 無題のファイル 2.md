CLIの[ホーム]画面に`セッション`を追加する
1キーで画面遷移する
startAtとstopAtは省略し、サーバータイムを使う。クライアントとの軽微なズレは許容する。
経過時間と残り時間はクライアント側で計算し、1秒毎に経過時間と残り時間の表示を更新したいんだけど、できる？デメリットはある？教えて
## 実行中セッションがある場合
```
[ホーム]
2026-03-10（火）01:01 WORK
Running: task 1h 12m 1s
Enter) セッション
1) プロトタイプ
2) オフ日管理
q) 終了
```
Enterで[セッション > 開始]へ

```
[セッション > 開始]
2026-03-10（火）14:32 WORK
名前: task
開始: 13:20:00
経過: 1h 12m 1s
残り: 10h 0m 59s
#focus #spec
https://example.com/task
https://example.com/hoge
Enter) 停止
p) プロトタイプ詳細
t) デイリータスク詳細（スタブ）
e) 開始時刻を編集
b) 開始前に戻る
h) ホームへ戻る
q) 終了
```
名前と経過はaccent color
タグとURLはmuted color

eで[セッション > 開始 > 編集]へ
Enterで[セッション > 開始 > 停止]へ

bを押したらまず確認を入れる。[プロトタイプ  >  詳細]のdを参考にする
y で DELETE /sessions/:id して[セッション]へ

p,t,h,qを押してもセッションは実行中のまま

```
[セッション > 開始 > 編集]
2026-03-10（火）14:32 WORK
名前: task
開始: 13:20:00  
経過: 1h 12m 1s  
残り: 10h 0m 59s
HHMM または HHMMSS で開始時刻を入力
空Enterで既存値を維持
```
名前と開始はaccent color
`HHMM または HHMMSS で開始時刻を入力`と
`空Enterで既存値を維持`はmuted color
入力後[セッション > 開始]に戻る

```
[セッション > 開始 > 停止]
2026-03-10（火）14:32 WORK
名前: task  
開始: 13:20:00  
停止: 14:32:01
経過: 1h 12m 1s
残り: 10h 0m 59s
Enter) サーバーに送信
p) プロトタイプ詳細
t) デイリータスク詳細（スタブ）
e) 停止時刻を編集
b) セッション停止前へ戻る
h) ホームへ戻る
q) 終了
```
名前と停止はaccent color
`Enter) サーバーに送信` はwarning color

eで[セッション > 開始 >  停止 > 編集]へ  
Enterで[セッション > 開始 > 停止]へ
p,t,h,qを押してもセッションは実行中のまま

```
[セッション > 開始 > 停止 > 編集]
2026-03-10（火）14:32 WORK
名前: task  
開始: 13:20:00  
停止: 14:32:01  
経過: 1h 12m 1s  
残り: 10h 0m 59s
HHMM または HHMMSS で停止時刻を入力
空Enterで既存値を維持
```

名前と停止はaccent color  
`HHMM または HHMMSS で停止時刻を入力`と  
`空Enterで既存値を維持`はmuted color
入力後[セッション > 開始 > 停止]に戻る

## 実行中セッションが無い場合
```
[ホーム]  
2026-03-10（火）01:01 WORK  
Next: task 15:00  
Enter) セッション
1. プロトタイプ
2. オフ日管理  
q) 終了
```
Enterで[セッション]へ

```
[セッション]  
2026-03-10（火）14:32 WORK  
名前: task
開始予定: 15:00
見積もり: 20m
https://example.com/task
https://example.com/hoge
Enter) 開始
p) プロトタイプ詳細
t) デイリータスク詳細（スタブ）
b/h) ホームへ戻る  
q) 終了
```
名前と開始予定と見積もりはaccent color  
タグとURLはmuted color
`Enter) 開始`はwarning color

---

```
Next: なし
1. プロトタイプ
2. オフ日管理  
q) 終了
```

```
[セッション]  
2026-03-10（火）14:32 WORK  
次のセッションはありません
p) プロトタイプ一覧
t) デイリータスク一覧（スタブ）
h) ホームへ戻る  
q) 終了
```
