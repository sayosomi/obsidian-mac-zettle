# キーの違い
- `⌘` = Win
- `⇧` = Shift
- `⌥` = `option` = alt
- `^` = `control` = ctrl 
# Finder
## パスをコピー
`option`（alt）を押しながら右クリックすると出る
## リネーム
ファイルを選択して`Enter`
# スクリーンショット
- **全画面**：`⌘ + ⇧ + shift + 3`
- **範囲選択**：`⌘ + shift + ctrl + 4`
- テンキーではない方の数字を押すこと
- クリップボードに入る
## ファイル削除
- `⌘ + Delete`
- 直接削除ではなくゴミ箱に入る
## カット & ペースト
- コピペ：`⌘ + C` → `⌘ + C`
- 移動：`⌘ + C` → `⌘ + shift + C`
## パスを確認
Finderのウィンドウからフォルダをターミナルにドラッグするとパスに変換される
## テンキーが反応しなくなったら
`⌘ + control + Q`でロックした後、ロック解除すると治る
## WindoswのDeleteキーの消し方
 `Fn + delete`
