# キーの違い
- `⌘` = Win
- `⇧` = Shift
- `⌥` = Alt
- `^`+Ctrl

# Finder
## パスをコピー
`⌥`（alt）を押しながら右クリックすると出る
## リネーム
ファイルを選択して`Enter`
# スクリーンショット
- **全画面**：`⌘ + ⇧ + Ctrl + 3`
- **範囲選択**：`⌘ + ⇧ + Ctrl + 4`
- テンキーではない方の数字を押すこと
- クリップボードに入る
## ファイル削除
- `⌘ + Delete`
- 直接削除ではなくゴミ箱に入る
## カット & ペースト
- コピペ：`⌘ + C` → `⌘ + C`
- 移動：`⌘ + C` → `⌘ + ⇧ + C`
## パスを確認
Finderのウィンドウからフォルダをターミナルにドラッグするとパスに変換される
