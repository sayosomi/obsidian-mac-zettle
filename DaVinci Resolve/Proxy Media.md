## 設定

[[Optimized Media and Render Cache]]

## [[Proxy Media]] を作る & 使う

- [[Source Clip (Media Pool)|Source Clip]] を右クリック > Proxy Media > Generate Proxy Media
- Playback > Proxy Handling > Prefer Proxies
		またはタイムラインビュー上のアイコンから ![](https://i.gyazo.com/efa1c197464087d489d8a99d7ec77c1c.png)

## Proxy Media を消す

- まず [[Source Clip (Media Pool)|Source Clip]] を右クリック > [[Proxy Media]] > Unlink Proxy Media
- その後 Finder で消す
	- 場所は [[Preferences]] > Working Folders から開ける

結構容量を食うので溜まってきたら消すこと
