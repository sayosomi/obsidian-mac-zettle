- メディアプールにドラッグアンドドロップで素材を追加
- 素材を右クリックして「選択したクリップで新規タイムラインを作成」
- `スペース`で再生
- `command + B`でクリップを分割
- 要らないクリップを選択して`delete`で削除
- 空白を右クリックで隙間を詰める
- 画面右上にある[[Inspector]]ボタンを押すと左側にパネルが出て選択したアイテムの移動やズーム、回転などができる
- [[横長の素材を正方形の縦幅いっぱいにトリムする]]
- タイムラインの先頭に移動する：`fn + ←` または `Home`
- タイムラインの末尾に移動する：`fn + →` または `End`
- `option + command + K`でキーボードショートカット設定画面を開く
