- メディアプールにドラッグアンドドロップで素材を追加
- 素材を右クリックして「選択したクリップで新規タイムラインを作成」
- `スペース`で再生
- `command + B`でクリップを分割
- 要らないクリップを選択して`delete`で削除
- 空白を右クリックで隙間を詰める
- 画面右上にある[[Inspector]]ボタンを押すと左側にパネルが出て選択したアイテムの移動やズーム、回転などができる
- [[横長の素材を正方形の縦幅いっぱいにトリムする]]
- タイムラインの先頭に移動する：`fn + ←` または `Home`
- タイムラインの末尾に移動する：`fn + →` または `End`
- `option + command + K`でキーボードショートカット設定画面を開く
- 音声を選んで`F10`でタイムラインに追加するとよい
- キーボードショートカットがおかしいときは日本語IMEをオフにしろ。GoogleIMEの英語入力は罠だ。Mac標準の英語入力にしないと詰む。
- 「カット」にいるのか「エディット」にいるのか常に意識する
- `Alt + マウスホイール`でタイムラインを拡大縮小
- `N`キーでスナップ（磁石アイコン）のON/OFFを切り換える。
- `L`で再生、`K`で停止。 `L`と`スペース`の違いは、`L`は押すたびに加速すること
- `command + shift`を押しながらクリップをドラッグすると、周りのクリップの長さに影響することなくクリップの順番を入れ替えることができる
